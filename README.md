# Openly

A calmer space for independent creators to blog, vlog, and host discussions.

## Quick Start

```bash
git clone https://github.com/your-org/openly.git
cd openly
pnpm install
cp .env.example .env
pnpm db:migrate
pnpm dev
```

Web: http://localhost:3000 | API: http://localhost:4000

## Stack
Next.js 14 + React Native (Expo) + Fastify + PostgreSQL + Stripe + Turborepo

See docs/ for full documentation.
